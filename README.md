# Raskladka Fix 🇺🇦🔄🇺🇸

Резидентна утиліта (Tray App) для **Windows 10/11**, яка дозволяє миттєво виправляти текст, набраний не в тій розкладці клавіатури (наприклад, англійські «крякозябри» замість українських літер і навпаки).

Достатньо просто виділити текст у будь-якому додатку та натиснути комбінацію клавіш (**Ctrl+Shift+U** за замовчуванням).

---

## ✨ Основні можливості

- **Миттєве виправлення:** Працює глобально в усій системі (Word, Chrome, Slack, Discord, Notepad тощо).
- **Розумна конвертація:** Автоматично визначає потрібний напрямок перекладу розкладки або дозволяє зафіксувати його (EN ↔ UA).
- **Візуальний візард налаштувань:** Красивий та інтуїтивний UI для кастомізації хоткеїв, напрямку конвертації, увімкнення автозапуску та сповіщень.
- **Два формати інсталяції:** Зручний `.msi` інсталятор або автономний `.exe` (NSIS).
- **Приємні дрібниці:** 
  - Розумне очищення модифікаторів клавіатури перед імітацією вставки (запобігає випадковому виклику DevTools у браузерах чи форматування у MS Word).
  - Можливість автозапуску разом із Windows.
  - Інтуїтивна іконка в системному треї (жовта та блакитна стрілочки по колу).

## 🛠 Технологічний стек

Застосунок розроблено за допомогою сучасних та швидких технологій:
- **Tauri 2.x** (Rust + WebView2) — забезпечує надзвичайно низьке споживання оперативної пам'яті (до 20-30 МБ) та миттєвий відгук.
- **React + Vite + Tailwind CSS** — сучасний та легкий інтерфейс для вікна налаштувань.
- **Rust (Enigo, Global Shortcut, Autostart)** — надійне низькорівневе перехоплення гарячих клавіш та імітація введення.

## 📥 Завантаження та Встановлення

Ви можете знайти готові до використання інсталятори на сторінці **[Releases](https://github.com/basermax-create/ukrainian-keyboard-fixer/releases)**.

1. Завантажте файл `.msi` (рекомендовано) або `.exe`.
2. Запустіть його та виконайте прості інструкції інсталятора.
3. *Примітка:* При першому запуску Windows SmartScreen може попередити про «невідомого видавця» — це стандартна поведінка системи для некомерційних збірок без дорогого платного цифрового підпису. Просто натисніть **Детальніше (More info)** → **Виконати в будь-якому разі (Run anyway)**.

## 🚀 Як користуватися

1. Переконайтеся, що застосунок запущено (у правому нижньому кутку екрана у треї з'явиться кругла жовто-блакитна іконка).
2. Виділіть текст, набраний помилково не в тій розкладці (наприклад: `ghbdtn, hfcyf rjycjk` або `ирзув, фцуіуьу`).
3. Натисніть **Ctrl + Shift + U** (або ваш власний налаштований хоткей).
4. Текст буде автоматично замінено на правильний: `привіт, красна консоль` чи `speed, awesome`.
