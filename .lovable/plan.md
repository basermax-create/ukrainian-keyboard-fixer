# План: Резидентна утиліта конвертації розкладки для Windows

## Концепція
Легка програма, що висить у системному треї Windows і чекає глобальну гарячу клавішу. Користувач виділяє «крякозябри» (ghbdsn) у будь-якому додатку → натискає хоткей (за замовч. `Ctrl+Shift+U`) → текст миттєво замінюється на «привіт».

## Платформа
**Тільки Windows 10/11 (x64)**. macOS виключаємо повністю — економія часу, грошей ($99/рік Apple Developer + нотаризація) та підтримки.

## Технологічний стек
- **Tauri 2.x** (Rust + WebView2) — ~5–10 MB інсталятор, нативні глобальні хоткеї, низьке споживання RAM (~30–50 MB)
- **UI**: React + TypeScript + Tailwind (мінімальне вікно налаштувань)
- **Плагіни Tauri**: `global-shortcut`, `clipboard-manager`, `autostart`, `tray-icon`, `notification`
- **Емуляція клавіш**: crate `enigo` (SendInput під капотом)

## Архітектура роботи
```text
[Користувач виділяє "ghbdsn" в Word/Chrome/Telegram]
            ↓
[Натискає Ctrl+Shift+U]
            ↓
Rust backend:
  1. Зберігає поточний вміст clipboard
  2. Емулює Ctrl+C → читає виділений текст з clipboard (затримка 50мс)
  3. Прогоняє через мапінг QWERTY → ЙЦУКЕН
  4. Кладе результат у clipboard
  5. Емулює Ctrl+V → текст вставляється замість виділеного
  6. Відновлює оригінальний clipboard (через 200мс)
  7. Показує toast-нотифікацію "Конвертовано"
```

## Ядро конвертації
Чиста статична таблиця відповідностей (без словників, без ML):
- Літери: `q→й, w→ц, e→у, r→к, t→е, y→н, u→г, i→ш, o→щ, p→з, [→х, ]→ї, a→ф, s→і, d→в, f→а, g→п, h→р, j→о, k→л, l→д, ;→ж, '→є, z→я, x→ч, c→с, v→м, b→и, n→т, m→ь, ,→б, .→ю`
- Shift-варіанти для великих літер
- Спецсимволи: `` ` →' ``, `~→Ё` (опц.)
- Двосторонньо (ЙЦУКЕН → QWERTY теж, для випадків навпаки) — кнопка в треї для зміни напрямку

## UI
**Tray icon** (іконка біля годинника):
- Конвертувати виділене (хоткей)
- Напрямок: UA↔EN / EN↔UA / Авто
- Налаштування...
- Автозапуск з Windows ☑
- Про програму
- Вихід

**Вікно налаштувань** (відкривається з трея):
- Зміна гарячої клавіші (capture combo)
- Режим: «замінити виділене» / «тільки в буфер»
- Показувати нотифікації ☑
- Автозапуск ☑
- Затримка clipboard (advanced, 50–200мс)

## Етапи розробки

**Етап 1 — PoC (1 день)**
AutoHotkey v2 скрипт як швидкий прототип, щоб перевірити сценарій з clipboard в реальних додатках (Word, Chrome, Telegram, VSCode, термінал).

**Етап 2 — MVP Tauri (5–7 днів)**
- Базовий tray + хоткей + конвертація
- Без вікна налаштувань (хоткей хардкоднутий)
- Локальна збірка `.exe`

**Етап 3 — v1.0 (1–2 тижні)**
- Вікно налаштувань
- Автозапуск
- MSI/NSIS інсталятор
- Підпис коду (Authenticode сертифікат, ~$80–200/рік) — щоб Windows SmartScreen не лякав користувача
- Auto-updater (Tauri updater + GitHub Releases)

**Етап 4 — v1.x (за потребою)**
- Історія конвертацій (останні 20)
- Підтримка RU↔EN (опційно)
- White-list додатків (не спрацьовувати у Password Manager, VM)
- Темна тема налаштувань
- Лічильник «врятованих секунд»

## Ризики та обмеження
| Ризик | Мітигація |
|-------|-----------|
| Деякі додатки блокують Ctrl+C з виділення (поля паролів, термінали) | Toast «Не вдалось прочитати виділення», fallback — показати конвертований текст у віконці |
| Google Docs / Notion з кастомним clipboard | Тестування + документація відомих обмежень |
| Windows SmartScreen блокує непідписаний `.exe` | Code signing сертифікат на етапі v1.0 |
| Антивіруси можуть реагувати на емуляцію введення | Підпис + поступове накопичення репутації |
| Затримка clipboard непередбачувана | Налаштовувана затримка + retry-логіка |

## Дистрибуція
- GitHub Releases: `.msi` + `.exe` (NSIS)
- Опційно: лендінг на Lovable з кнопкою «Завантажити для Windows»
- Опційно пізніше: Microsoft Store (потребує реєстрації розробника $19 одноразово)

## Що НЕ робимо
- ❌ macOS версію (відмовляємось повністю)
- ❌ Linux версію (можна додати пізніше — Tauri підтримує безкоштовно, але поза скоупом)
- ❌ Мобільні версії
- ❌ Хмарну синхронізацію, акаунти, бекенд
- ❌ Словникову/LLM корекцію орфографії (тільки чистий мапінг розкладки)
